                                                                            Carlo van Wyk Mod Samuel Shirley • 8 months ago

                                                                            in this context, options.Cookie.HttpOnly = true prevents the browser from making the cookie available to client side scripts, which is an added security feature. HttpOnly doesn't refer to HTTPS in this context. (There's no reason any client side scripts should have access to the cookie.)

                                                                            HTTPS is set through options.Cookie.SecurePolicy = CookieSecurePolicy.Always

                                                                                                      Carlo van Wyk Mod JamieKeeling • a year ago

                                                                                                      Good question. I wrote an article about managing secrets here : https://www.humankode.com/a...

                                                                                                      To keep the certificate password out of source control, simply do the following :
                                                                                                      1) Create a certificate.json as per example in tutorial, but leave the password blank
                                                                                                      2) Create a certificate.development.json and store your password in this file
                                                                                                      3) Chain the certificate.development.json up in your program.cs as per the example in this tutorial
                                                                                                      4) Add a .gitignore rule to ignore certificate.development.json so that your password isn't checked into source control
